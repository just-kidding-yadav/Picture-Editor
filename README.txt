This project contains 2 files- pgm.py and edit.py
The "pgm.py" file provides functions readpgm to read a .pgm file and convert it into array
and writepgm to convert an array into a .pgm file.
The file "edit.py" provides functions to edit an array (corresponding to a pgm file):
1. averaging
2. edge_detection
3. normal (to normalise the edges)
4. threshold
5. minEnergy: draws a minimum energy path
